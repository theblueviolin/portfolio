Good Morning

 

Summer 2025

Message Generator Database Website

Javascript
HTML
CSS
Has your girlfriend/boyfriend/partner ever gotten mad at you for not texting good morning? Well look no further. The Good Morning Message Generator is a lightweight web app designed to automate personalized good morning texts for loved ones. Built with HTML, CSS, and JavaScript, the app allows users to select a message style, define a time range, and send the message via iMessage with one click. To streamline the experience, users can also save frequently used phone numbers locally in the browser.

This project was both a fun and practical way to explore frontend web development and user interaction design. It deepened my understanding of DOM manipulation, mobile-first UX principles, and integrating device-specific functionality (like SMS/iMessage link schemes).

Memory Matching Game

 

Spring 2023

Interactive Matching Game Website

HTML
CSS
Javascript
Browser APIs
Web Storage (localStorage)
Memory Matching Madness is a browser-based game where players test their memory by matching hidden Pokémon images within a time limit. Built with HTML, CSS, and JavaScript, the game offers multiple difficulty levels and shuffles the image tiles randomly each time you play, ensuring a unique experience. Features like a real-time timer, sound effects for correct and incorrect matches, and a local high-score tracker using localStorage make the game interactive and replayable.

This project helped me strengthen my understanding of DOM manipulation, event-driven programming, and game logic design. Implementing the randomization logic for the tiles, dynamically generating the board based on difficulty, and managing game state without libraries provided hands-on experience with core JavaScript concepts and front-end development practices.
